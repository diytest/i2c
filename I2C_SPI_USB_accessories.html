Планирую разместить:

1) разные платы с одной микросхемой, которые можно подключить к адаптеру.
Из того, что уже опробовано и есть ПО:
  акселерометры, SPI интерфейс - ADXL345(16g), ADXL375(200g), ADXL355 (8g прецизионный), BMA280(16g, низкий шум), LIS3DH
  память на 256КБайт, SPI интерфейс
  высокоточные датчики влажности, давления, температуры, I2C интерфейс
  датчик температуры ходовой DS18B20, 1-Wire интерфейс
  часы реального времени I2C
  7-сегментные индикаторы       
Вид плат примерно такой, как на картинке отдельные платы выше платы адаптера в корпусе.
Для каждой платы предоставлять прошивки для адаптеров.

2) можно предлагать программаторы с aliexpress.для разных микроконтроллеров, примененных в адаптерах.
  Сейчас 3 микроконтроллера - для них 3 разных программатора.
  Программатор для одного из них на картинке (верхняя плата).
3) корпуса для адаптеров и кабельные вводы под корпуса - также с aliexpress..
4) отдельная статья - кабели для подключения плат к адаптеру, это хлопотно и затратно, т.к. изготовление кабелей и монтаж выполняются вручную.

Платы, подключаемые к адаптеру I2C SPI 1-Wire USB

Память FRAM 256Кх8 FM25V20A, 8 контактов, SPI

разместить фото отдельной платы с ИС памяти, отдельно плату без микросхемы - можно предлагать плату с памятью или пустую плату как дополнение к адаптеру.

Такая же информация для других устройств,

для которых уже есть или появятся примеры (см. ПО и библиотека примеров) - для каждого устройства отдельно предлагать плату с ИС и пустую плату.
Кому-то будет проще взять адаптер+устройство с наценкой, чем отдельно заказывать на aliexpress.

Такая же информация по корпусу с ali

его тоже предлагать с наценкой - можно купить на alibaba.com, предлагать по цене aliexpress за единичный экземпляр - у них соотношение цен 4:1

Программатор ST-Link для адаптера с STM8, возможно, и программатор для ATmega8 с ali

предлагать с наценкой - можно купить на ali + наценка; если пересылать вместе с адаптером, стоимость пересылки вряд ли увеличится - вес и габариты малы.

Можно предлагать кабели под заказ или приобрести готовые 

если это будет выгодно: делать мягкие и легкие кабели для акселерометров, гироскопов - устройств, требующих минимального веса платы с датчиком и максимальной лёгкости и гибкости кабеля; от покупных кабелей можно отрезать необходимый заказчику кусок;
можно делать комплект под ключ - адаптер, кабель, амортизатор, корпус, выбранные платы с устройствами.
Всё это вопрос цены и есть ли кому заниматься монтажом и на каких условиях.
Скорее всего, лучше пока ограничиться адаптерами, а в дальнейшем, возможно, отправкой наборов: адаптер + корпус + платы устройств с микросхемами или без микросхем, не связываясь с кабелями вообще.
